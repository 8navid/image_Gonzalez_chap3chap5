# Image Degradation and Restoration

A comprehensive Python implementation of image degradation models and restoration techniques based on **Digital Image Processing** by Gonzalez & Woods (Chapter 5).

---

## 📚 Overview

This project demonstrates:
- **Image Degradation**: Simulate blur (Gaussian/Motion) and additive Gaussian noise
- **Image Restoration**: Apply three frequency-domain filters:
  - Inverse Filter
  - Wiener Filter (Minimum Mean Square Error)
  - Constrained Least Squares Filter
- **Noise Models**: Gaussian, Salt & Pepper, Uniform, Exponential
- **Enhancement**: Contrast Stretching, Histogram Equalization, Mean Filter

---

## 🚀 Features

| Module | Description |
|--------|-------------|
| `noise.py` | Generate and visualize 4 types of noise |
| `degres.py` | Apply Gaussian/Motion blur + noise and Inverse, Wiener, and Constrained Least Squares filters |
| `enhance.py` | Contrast stretching, histogram equalization, mean filter |
